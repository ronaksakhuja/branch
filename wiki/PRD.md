# Branch PRD

**Tagline:** Git for documents. Built for humans and AI.

Branch is a markdown workspace with version history, local CLI sync, and AI collaboration.
Humans and AI edit the same source of truth with full change tracking and rollback.

## Core Product

- Cloud markdown workspace
- Version history with diffs
- CLI for local editing and sync
- AI-ready CLI and API
- Google Docs-like web UI

## MVP Status

- Web app deployed on Vercel with Clerk auth and Neon Postgres
- CLI published as `getbranch` on npm
- Full CRUD with version history
- CLI auth via browser redirect

## Long Term

GitHub became the home for code. Branch becomes the home for knowledge.
